# asr-imagery-list

The asr-imagery-list repository contains all imagery-related resources, including JSON schemas, sample imagery JSON files, and imagery configuration data.

# Schema:

Defines the structure and validation rules for imagery JSON files. This schema is used by both the Workspaces frontend and backend to ensure consistency and correctness of imagery data.

# Example:

Provides ready-to-use sample JSON files for quick reference, helping developers easily create or validate new imagery configurations.

# Imagery:

Maintains a centralised record of all imagery data used across different Workspaces, ensuring version control, traceability, and alignment between environments.

Overall, this repository serves as the single source of truth for all imagery-related definitions and configurations used in the Workspaces ecosystem.
